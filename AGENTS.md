# Rafael Susano — Portfolio

Personal portfolio built with TanStack Start, React, TypeScript, and Tailwind CSS v4.

## Tech Stack

- **Framework**: TanStack Start (SSR)
- **Styling**: Tailwind CSS v4
- **Animations**: Motion (Framer Motion v12)
- **Icons**: Lucide React, React Icons
- **Fonts**: Space Grotesk, JetBrains Mono, Instrument Serif

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
